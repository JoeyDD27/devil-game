# üöÄ Quick Start Guide

Get your Devil Game extension running in 3 easy steps!

## Step 1: Create the Icons

**Option A - Use the Icon Generator (Recommended):**
1. Open `create-icons.html` in your web browser (just double-click it)
2. Click the download button under each icon
3. Save them as `icon16.png`, `icon48.png`, and `icon128.png` in this folder

**Option B - Use Online Tools:**
1. Go to any free icon generator (like flaticon.com or favicon.io)
2. Create three icons: 16x16, 48x48, and 128x128 pixels
3. Use a Halloween theme (pumpkin, ghost, devil, etc.)
4. Save them in this folder with the names above

**Option C - Quick Placeholder:**
1. Create any three small PNG images (screenshots work!)
2. Resize them to 16x16, 48x48, and 128x128
3. Name them correctly and place in this folder

## Step 2: Load the Extension

1. Open Google Chrome
2. Go to: `chrome://extensions/`
3. Turn ON "Developer mode" (toggle switch in top-right corner)
4. Click "Load unpacked" button
5. Select this "devil game" folder
6. Done! ‚úÖ

## Step 3: Play!

1. Look for the Devil Game icon in your Chrome toolbar (top-right)
2. Click it to start playing
3. Make your choices and discover different endings!
